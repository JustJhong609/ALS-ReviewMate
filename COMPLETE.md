```bash
cd /workspaces/ALS-ReviewMate

```

### 2. Set Up Database (10 minutes)
1. Go to [supabase.com](https://supabase.com)
2. Create account and new project
3. Copy `supabase-schema.sql` content
4. Paste in SQL Editor and run
5. Optionally run `sample-data.sql` for test data

### 3. Test Locally (15 minutes)
```bash
npm run dev
```
- Visit `http://localhost:3000`
- Register as learner and teacher
- Test quiz functionality
- Check progress tracking
- Verify teacher dashboard

- Open: `npx cap open android`
